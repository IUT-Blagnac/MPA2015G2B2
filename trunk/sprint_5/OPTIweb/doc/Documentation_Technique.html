Table des matières

OPTIweb.html est une application HTML de consultation des projets tutorés gérés par OPTI.

OPTIweb.html est non compatible avec le navigateur Google Chrome mais l’est pour les supports du moment (PC, tablettes, smartphones).

Application

  • En tant qu’internaute, je veux accéder à l’application OPTIweb depuis tout support fixe ou mobile

  • En tant qu’internaute, je veux pouvoir consulter les projets, les sujets, les étudiants et les intervenants dans des pages séparées

  • En tant qu’internaute, je veux pouvoir consulter le nom des développeurs de l’application, le nom du Product Owner ainsi que les technologies utilisées

Page Projets

  • En tant qu’internaute, je veux pour chaque projet en voir le nom, le titre, le client, le superviseur, le groupe et les étudiants réalisant le projet

  • En tant qu’internaute, je veux pouvoir rechercher un projet dans la liste des projets en fournissant une chaine de caractère quelconque

Page Sujets

  • En tant qu’internaute, je veux pour chaque sujet en voir le nom, le titre et le ou les noms des groupes d'étudiants réalisant le sujet

  • En tant qu’internaute, je veux pouvoir rechercher un sujet dans la liste des sujets en fournissant une chaine de caractère quelconque

  • En tant qu’internaute, je veux pouvoir sélectionner un sujet et voir les projets réalisant le sujet

Page Etudiants

  • En tant qu’internaute, je veux pour chaque étudiant en voir le nom, le prénom ainsi que son groupe

  • En tant qu’internaute, je veux pouvoir rechercher un étudiant dans la liste des étudiants en fournissant une chaine de caractère quelconque

  • En tant qu’internaute, je veux pouvoir sélectionner un étudiant et voir le projet qu’il réalise

Page Intervenants

  • En tant qu’internaute, je veux pour chaque intervenant en voir le nom, le prénom ainsi que le nombre de ses interventions en tant que Client ou Superviseur

  • En tant qu’internaute, je veux pouvoir rechercher un intervenant dans la liste des intervenants en fournissant une chaine de caractère quelconque

  • En tant qu’internaute, je veux pouvoir sélectionner un intervenant et voir le(s) projet(s) où il intervient

Comment lancer l’application OPTIweb

  1. Sur tout support, lancer le fichier OptiWeb.html dans un navigateur

Note

L’application OPTIweb.html nécessite une connexion Internet lors de son lancement.

Une fois chargée dans un navigateur, aucune communication réseau n’est éffectuée : vous pouvez alors travailler hors connexion.

Comment sauvegarder l’application OPTIweb

  1. Enregistrer le fichier OPTIweb.html sur votre support (environ 30Ko)

  2. Charger le fichier OPTIweb.html local dans un navigateur pour lancer l’application (une connexion réseau n’est requise qu’au chargement).

Génération du fichier OPTIweb.html

L’application de consultation OPTIweb.html est intégralement générée par le programme MakeOPTIweb.java qui à partir des fichiers csv (Sujets, Etudiants, Intervenants, Projets …) fabrique le fichier OPTIweb.html. Ce programme crée le fichier html en concaténant les balises du code html (sous forme de String) avec les valeurs des fichiers csv.

Note
 Le programme MakeOPTIweb.java utilise la librairie de gestion des fichiers csv disponible dans le projet OPTI.

Organisation du fichier OPTIweb.html

Le fichier OPTIweb.html produit par le programme MakeOPTIweb.java possède la structure html/jquery/mobile suivante :

  1. css font-awesome

  2. css jquery-mobile

  3. javascript jquery

  4. javascript jquery mobile

  5. css de gestion des classes d’orientation du support

  6. Les pages #accueil, #credits, #projets, #sujets, #etudiants et #intervenants sont placées ici

  7. handler jquery de sélection des projets depuis les pages #sujets, #etudiants ou #intervenants

La sélection des projets correspondants à l’item choisi des pages #sujets, #etudiants ou #intervenants s’effectue ainsi :

  1. L’action click sur les items (li) comportant 1 attribut data-find

  2. exécute le code (l’handler) suivant :

    qui place le contenu actuel de l’attribut data-find dans le champs input de recherche des projets d’id autocomplete-input-projet, puis valide la modification.

  3. L’affichage de la page #projets qui résulte de l’action standard sur l’item, n’affichera que les projets sélectionnés.

OPTIWeb : Test Casper

Casper permet de tester les différents aspect de l’application HTML :

  • casperAccueil

  • casperCredits

  • casperEtudiants

  • casperIntervenants

  • casperProjets

  • casperSujets

Par exemple, lors de ces test, nous regardons si les sujets de l’interface sont ceux que nous avons importé. Aussi, nous vérifions les réactions de l’application lorsque nous cliquons sur différents boutons. Par exemple, lorsque nous cliquons sur le boutton "crédits", des tests sont alors réalisés pour vérfier si l’affichage des informations est correct. Il y a plus de 200 insertions pour l’application OptiWeb. Les tests sont lancés à partir du fichier casperjs.bat qui est dans le repertoire test à la racine du sprint 4.

OptiWeb : Test MakeOPTIWebTest

Le programme de test MakeOPTIWeb.java vérifie si le fichier a bien été généré et si le contenu du fichier OptiWeb.hmtl est bien égal à ce que l’on souhaite obtenir.
