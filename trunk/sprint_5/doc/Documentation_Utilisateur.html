4. Présentation de l’interface et des fonctionnalités de base

4.1. Interface de base

L’interface se présente en plusieurs parties bien disctinctes, dont nous allons voir l’organisation ici.

Figure 1. Interface de l’application

Après avoir ouvert l’application, vous pouvez effectuez plusieurs action de base présentez juste après.

4.2. Fonctionnalités de base : le menu Fichier

Figure 2. Fichier

Dans ce menu on peut accéder à trois fonctionnalités :

  • Ouvrir fichier csv :

Figure 3. Fenetre d’ouverture du fichier csv

  • Sauvegarder CSV : pemet de sauvegarder un fichier

L’ouverture d’un fichier csv permet de generer toute la base de données contenant les sujets. Ceci n’est pas indispensable à la création de nouveaux sujets, ainsi qu'à la gestion d’un nouveau projet.

  • A propos :

Figure 4. A propos

Cette fenetre detaille les réalisateurs de l’application ainsi que quelques informations sur la version.

  • Quitter :

Figure 5. Quitter

Ouvre une fenetre demandant la confirmation pour quitter l’application. Dans le cas où vous selectionnez "Oui", l’application se ferme, sinon la fenetre se ferme et vous pouvez continuer à utiliser normalement l’application.

5. Description des différentes fonctionnalités

Une fois un fichier csv ouvert, l’onglet sujet se rempli de sujets et on accède à différentes fonctionnalités.

5.1. Onglet Sujets

Figure 6. L’onglet Sujets au sein de l’interface graphique

Cet onglet, dans la partie gauche, permet de visualiser la liste des sujets. La partie droite de l’onglet affiche le sujet selectionné en plusieurs partie. La partie inferieur contient des boutons permettant les differentes actions : ajout, modification et suppression de sujets.

5.1.1. Partie visualisation

Figure 7. Entete de la partie visualisation

  • Acronyme : nom du sujet plus cours afin de le reperer plus facilement

  • Titre : titre entier du sujet

  • Onglet Contexte : décrit le contexte du projet, l’histoire de certaines entreprises afin de se reperer pour mieux comprendre le sujet

  • Onglet Projet : décrit précisement les objectifs attendus

  • Onglet Correspondant : contient les correspondant ainsi que leurs coordonnées en lien avec ce sujet

Figure 8. Outils, situé en bas de la partie de droite

  • Outils : renseigne les outils nécessaire au projet, ainsi que ceux qui sont conseillés

5.1.2. Partie liste

Dans cette partie, on peut visualiser la liste des sujets.

5.1.3. Partie inférieur

Figure 9. Boutons du bas de la page

Voici les actions réalisables avec ces boutons :

  • Ajout d’un nouveau sujet :

Figure 10. Ajout d’un nouveau sujet

Après le remplissage de ce formulaire, un nouveau sujet sera créer au sein de la base de données.

  • Modification d’un sujet

Figure 11. Modification d’un sujet

Ce bouton active la modification d’un sujet, et on peut modifier directement au sein de la partie visualisation

  • Supprimer un sujet

Figure 12. Suppression d’un sujet

Ce bouton permet de supprimer un sujet en selectionnant le sujet pour le visualiser, le bouton est grisé si aucun sujet n’est selectionné.

5.2. Onglet Etudiants

Figure 13. Onglet étudiants

Cet onglet se présente en plusieurs partie :

  • liste des groupes

    • bouton ajouter

  • etudiants dans le groupe

  • liste des etudiants

    • section de boutons ajouter, créer, modifier et supprimer

5.2.1. Liste des groupes

Le fait de cliquer sur le bouton "Creer" ajoute un groupe dans la liste. Le chiffre a gauche correspond au nombre d'étudiants dans le groupe et le groupe porte le nom de "groupe n", n allant de 1 au nombre de groupe de la liste.

5.2.2. Liste des étudiants

Dans cette partie on peut voir la liste des étudiants. On peut aussi en ajouter en cliquant sur créer.

Figure 14. création étudiants

Si on clique sur modifier, une fenetre assez similaire avec le nom de l'étudiant sélectionné s’ouvre et on peut modifier le nom et le prenom.

Si on clique sur ajout l’etudiant s’ajoute dans le groupe selectionné.

Si on clique sur supprimmer, l'étudiant ce supprime.

6. Ce que fais le SPRINT 3…

Notre SPRINT 3 permet d’ouvrir un fichier pour generer tous les sujets ainsi que de les supprimer. On peut également en créer des nouveaux, et les supprimer. On peut également visualiser les groupes et les étudiants dans l’onglet étudiant.
